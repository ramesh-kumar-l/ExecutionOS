# Principles — LENSSTACK + X10THINK

## 15 Mandatory Product Principles

1. **Offline-first** — all core functionality works without internet
2. **Local-first ownership** — user owns all data, no vendor lock-in
3. **AI as assistant, not controller** — AI suggests, human decides
4. **Calendar-first execution** — goals become scheduled time or they don't exist
5. **Deep work protection** — system actively shields focus time
6. **Minimal cognitive load** — every screen reduces friction, never adds it
7. **Strategic alignment over busyness** — measure outcomes not activity
8. **Simplicity over feature overload** — fewer, better features
9. **Reliability over flashy features** — trust through consistency
10. **Human trust above AI cleverness** — explainability over magic
11. **Long-term maintainability** — architect for decade-long use
12. **Explainable AI recommendations** — always show reasoning
13. **No manipulative UX** — no dark patterns, streaks pressure, or artificial urgency
14. **Elegant premium experience** — aesthetic quality signals care
15. **Systems thinking over motivation** — build systems not willpower

## Engineering Principles

- Correctness before performance
- Strong typing everywhere (TypeScript + Rust)
- Modular over monolithic
- Deterministic state transitions
- Fail loudly in development, gracefully in production
- No premature optimization
- No speculative abstractions
- Test behavior, not implementation

## AI Behavior Principles

- Never override user decisions
- Always explain recommendations
- Detect overload/burnout risk without alarming
- Maintain continuity — remember context over time
- Degrade gracefully when AI is unavailable
- No fake positivity or hype language
- No manipulation through urgency or gamification

## UX Principles

- Keyboard-first interactions
- Low friction above all else
- Fast perceived performance (optimistic UI)
- High information density without clutter
- Premium typography as primary visual element
- Smooth subtle animations (never distracting)
- Deep work oriented layouts
- Distraction-free modes

## Data Principles

- User owns 100% of data
- Encrypted at rest for sensitive fields
- No hidden telemetry
- Export at any time in open formats
- No forced sync or cloud dependency
